## Overview

本 Skill 专注 A 股连板（连续涨停）交易策略，提供涨停池筛选、连板高度统计、炸板率计算、封单金额分析等核心功能。适用于短线打板交易，支持主板、创业板、科创板。

**依赖**：
- 首选数据源：[amazingdata skill](../amazingdata/SKILL.md)（本地实时数据）
- 备选数据源：[tushare skill](../tushare/SKILL.md)（需 token）
- 备选数据源：[akshare skill](../akshare/SKILL.md)（免费，无 key）

## 适用范围

- **仅 A 股主板 / 创业板 / 科创板**
- 专注**短线连板交易**（持股周期 1-5 天）
- 不适用于：港股、美股、基金、可转债

## 触发条件

用户提问形如：
- "今天涨停池有哪些"
- "3连板以上的股票"
- "炸板率高的股票"
- "封单金额排名"
- "帮我筛选打板标的"

## 核心指标

### 1. 涨停池（Limit-up Pool）

当日涨停的股票集合。

| 字段 | 说明 |
|------|------|
| `code` | 股票代码 |
| `name` | 股票名称 |
| `limit_price` | 涨停价 |
| `first_time` | 首次涨停时间 |
| `last_time` | 最后封板时间 |
| `open_count` | 炸板次数（开板次数） |
| `volume` | 成交量 |
| `amount` | 成交额 |

### 2. 连板高度（Consecutive Days）

连续涨停天数。

| 等级 | 说明 |
|------|------|
| 首板 | 1天涨停（首次） |
| 2连板 | 连续2天涨停 |
| 3连板 | 连续3天涨停 |
| 4连板+ | 高标股，风险收益比变化 |

### 3. 炸板率（Open Rate）

```
炸板率 = 炸板次数 / (炸板次数 + 1) × 100%
```

- 炸板率 > 50%：封板不坚决，谨慎
- 炸板率 < 20%：封板坚决，强势

### 4. 封单金额（Seal Amount）

涨停价位的委买金额。

| 等级 | 封单金额 | 说明 |
|------|---------|------|
| 极强 | > 5亿 | 资金抢筹明显 |
| 强 | 2-5亿 | 封板坚决 |
| 中等 | 5000万-2亿 | 一般 |
| 弱 | < 5000万 | 容易炸板 |

### 5. 封单比（Seal Ratio）

```
封单比 = 封单金额 / 流通市值 × 100%
```

- 封单比 > 10%：极度强势
- 封单比 5-10%：强势
- 封单比 < 2%：偏弱

## 筛选策略

### 策略 A：强势连板（高风险高收益）

条件：
- 连板高度 ≥ 3天
- 炸板率 < 20%
- 封单金额 > 2亿
- 封单比 > 5%

### 策略 B：首板突破（中风险中收益）

条件：
- 首板（1天涨停）
- 前期横盘或突破形态
- 封单金额 > 5000万
- 所属板块有热度

### 策略 C：反包板（低吸策略）

条件：
- 前日炸板或冲高回落
- 今日反包涨停
- 炸板率 < 30%
- 成交量放大但不爆量

## 风险控制

### 仓位管理

| 连板高度 | 建议仓位 |
|---------|---------|
| 首板 | 20-30% |
| 2连板 | 15-20% |
| 3连板 | 10-15% |
| 4连板+ | 5-10% |

### 止损规则

- 开盘跌破昨日涨停价：立即止损
- 盘中炸板且 10 分钟内未回封：减仓 50%
- 收盘未封住：清仓

### 板块联动

- 关注同板块其他个股表现
- 龙头炸板时，跟风股优先离场
- 板块集体退潮日，不开新仓

## 禁止事项

1. 不在非交易时间推荐打板（9:25 集合竞价后才能判断）
2. 不推荐 ST/*ST 股票打板
3. 不推荐流通市值 < 20亿 的标的（流动性风险）
4. 不逆势操作（大盘暴跌日空仓）
5. 不满仓单一个股
6. 不在涨停板上加仓（只开新仓或止损）
